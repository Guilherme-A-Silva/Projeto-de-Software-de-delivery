## Métodos de leitura ##

1. Realizar Compras
2. Visualizar a rota de entrega, com tempo estimado.
3. Pedidos Pendentes
4. Entregas Pendentes
5. Consultar Itens
6. Listagem de Itens

## Métodos de cadastro ##

1. Cadastrar Cliente
2. Cadastar Empresa
3. Realizar Compras
4. Cadastrar Itens
5. Editar Estoque
6. Excluir Itens
7. Excluir Estoque


## Método Atualizacao ##

1. Atualizar Dados de Login Cliente
2. Atualizar Dados de Login Empresa
3. Atualizar Estoque
4. Fechar movimento do dia (Caixa)
